⚙️ Data Collection Workflow

End-to-end pipeline from field recording to heatmap visualisation

🎤
1. Record Sound Levels
Use smartphone decibel meter apps to measure ambient noise at target locations. Take readings for 60 seconds and log the Leq (equivalent continuous level).
Decibel X NIOSH SLM Android dBmeter
📡
2. Tag with GPS
Simultaneously log GPS coordinates (lat, lon) using device location. Capture metadata: time, date, weather, and location type (residential / commercial / industrial).
GPS Logger Google Maps API OsmAnd
📂
3. Upload as CSV
Export readings as structured CSV with columns: timestamp, latitude, longitude, dB_leq, dB_min, dB_max, location_type, city.
CSV Export Python pandas Google Sheets
🐍
4. Process with Python
Clean and validate data, compute Leq averages, remove outliers, classify zones by WHO thresholds, and export GeoJSON for mapping.
pandas geopandas folium scipy
🗺️
5. Generate Heatmap
Render interactive noise heatmaps using Leaflet.js with heatmap.js plugin. Deploy to browser — zoom into hotspot areas and filter by time or location type.
Leaflet.js heatmap.js Google Maps API
📋 Sample Python Processing Script
Python 
import pandas as pd
import folium
from folium.plugins import HeatMap

# ─── Load GPS-tagged decibel CSV ───
df = pd.read_csv('noise_readings.csv')

# ─── WHO Zone Classification ───
def classify_zone(db):
    if db < 55:   return 'Quiet'
    elif db < 65: return 'Moderate'
    elif db < 80: return 'Loud'
    else:         return 'Dangerous'

df['zone'] = df['dB_leq'].apply(classify_zone)
df['weight'] = (df['dB_leq'] - 40) / 60  # normalise for heatmap

# ─── Generate Leaflet Heatmap ───
m = folium.Map(location=[28.6139, 77.2090], zoom_start=12)
heat_data = df[['latitude', 'longitude', 'weight']].values.tolist()
HeatMap(heat_data, radius=15, max_zoom=13).add_to(m)
m.save('noise_heatmap.html')
print(f"Map saved — {len(df)} readings plotted")
